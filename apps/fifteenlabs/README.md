# sf-voice TypeScript browser example

Fastify server with a small browser UI for ingest, task polling, search, and
asset listing.

## setup

```bash
cp .env.example .env
# fill in SF_VOICE_API_KEY
```

## smoke check

```bash
bun run typecheck
```

## run

```bash
bun run dev
```

Open `http://localhost:3000`.
