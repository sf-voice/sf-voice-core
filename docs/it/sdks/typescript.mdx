`@sf-voice/media` — richiede un runtime con `fetch`, `FormData` e `Blob`.

Versione attuale: `0.1.1`. Fissala mentre l'API è in movimento.

## Installazione

<CodeGroup>
```sh pnpm
pnpm add @sf-voice/media@0.1.1
```

```sh npm
npm install @sf-voice/media@0.1.1
```

```sh bun
bun add @sf-voice/media@0.1.1
```
</CodeGroup>

## Creare un client

```ts
import { SfVoiceMedia } from "@sf-voice/media";

const client = new SfVoiceMedia({
  baseUrl: "https://api.sf-voice.com",
  apiKey: process.env.SF_VOICE_API_KEY!,
});
```

| Opzione | Tipo | Predefinito | |
| --- | --- | --- | --- |
| `baseUrl` | `string` | — | Obbligatorio. |
| `apiKey` | `string` | — | Obbligatorio. Inviato come `X-API-Key`. |
| `timeoutMs` | `number` | `30_000` | Timeout di abort per richiesta. |

## Ingestione

`ingest(request)` invia una registrazione e restituisce immediatamente un `task_id`.

### Da un URL

```ts
const { task_id } = await client.ingest({
  source: "url",
  asset_id: "call_001",
  asset_class: "customer_acme",
  url: "https://storage.example.com/calls/001.mp3",
  media_type: "audio",
  types: ["audio", "transcript"],
  metadata: { caller: "+14155550199" },
});
```

### Da S3

```ts
const { task_id } = await client.ingest({
  source: "s3",
  asset_id: "call_001",
  asset_class: "customer_acme",
  s3_key: "calls/customer_acme/001.mp3",
  media_type: "audio",
  types: ["audio", "transcript"],
});
```

### Da un file

```ts
const buffer = await file.arrayBuffer();

const { task_id } = await client.ingest({
  source: "file",
  asset_id: "call_001",
  asset_class: "customer_acme",
  file: buffer,
  filename: "001.mp3",
  content_type: "audio/mpeg",
  media_type: "audio",
  types: ["audio", "transcript"],
});
```

`file` accetta `Blob`, `ArrayBuffer` o `Uint8Array`.

### Campi comuni

| Campo | Obbligatorio | |
| --- | --- | --- |
| `asset_id` | Sì | Il tuo ID. Restituito nei risultati di ricerca. |
| `asset_class` | No | Gruppo logico. Limita la ricerca affinché i risultati non sconfinino oltre i confini. |
| `media_type` | No | `"audio"` o `"video"`. Rilevato automaticamente se omesso. |
| `types` | No | Superfici da indicizzare: `"audio"`, `"video"`, `"transcript"`. Predefinito: tutte. |
| `metadata` | No | Coppie piatte string/number/boolean per il tuo uso. |

### Risposta

```ts
type IngestResponse = {
  asset_id: string;
  task_id: string;
  status: "pending";
};
```

## Task

### Polling fino a quando è pronto

```ts
const task = await client.pollTask(task_id, {
  intervalMs: 2_000,   // ogni quanto controllare (predefinito 1_500)
  timeoutMs: 120_000,  // rinuncia dopo questo tempo (predefinito 120_000)
});

if (task.status === "failed") {
  throw new Error(task.error ?? "indexing failed");
}
```

`pollTask` lancia `SfVoiceMediaPollTimeoutError` se il task non raggiunge `ready` o `failed` entro il timeout.

### Controllo singolo

```ts
const task = await client.getTask(task_id);
// task.status: "pending" | "indexing" | "ready" | "failed"
```

### Forma del task

```ts
type Task = {
  task_id: string;
  asset_id: string;
  asset_class?: string;
  types: Array<"audio" | "video" | "transcript">;
  status: "pending" | "indexing" | "ready" | "failed";
  error?: string;
  created_at: string;
  completed_at?: string;
};
```

## Ricerca

`search(request)` restituisce corrispondenze con timestamp nelle registrazioni indicizzate.

### Per asset class

```ts
const { results } = await client.search({
  query: "customer asks about cancelling their subscription",
  asset_class: "customer_acme",
  types: ["transcript"],
  threshold: 0.7,
  limit: 10,
});
```

### Per asset specifici

```ts
const { results } = await client.search({
  query: "pricing question",
  asset_ids: ["call_001", "call_002"],
});
```

### Cercare in tutti gli asset indicizzati

```ts
const { results } = await client.search({
  query: "AI interrupted the caller",
  scope: "all",
});
```

<Warning>
  `scope: "all"` cerca in tutti i tuoi asset indicizzati. Usa `asset_class` o `asset_ids` per qualsiasi cosa rivolta ai clienti.
</Warning>

### Campi di ricerca

| Campo | Obbligatorio | |
| --- | --- | --- |
| `query` | Sì | Linguaggio naturale. |
| `types` | No | `"audio"`, `"video"`, `"transcript"`. Cerca in tutti se omesso. |
| `asset_class` | No | Restringere a un gruppo logico. |
| `asset_ids` | No | Restringere a un insieme specifico di asset. |
| `scope` | No | Imposta a `"all"` per cercare tra gli asset indicizzati. |
| `threshold` | No | `0.0`–`1.0`. Predefinito `0.5`. Più alto = meno risultati, più affidabili. |
| `page` | No | Numero di pagina. |
| `limit` | No | Risultati per pagina. Max 50. |

### Forma del risultato

```ts
type SearchResult = {
  asset_id: string;
  score: number;      // 0.0–1.0
  start_ms: number;   // offset nella registrazione
  end_ms: number;
  match_type: "audio" | "video" | "transcript";
  thumbnail_url?: string;
};
```

`start_ms` e `end_ms` sono millisecondi nella registrazione sorgente. Usali per posizionare un player sul segmento corrispondente.

## Asset

```ts
// elenco (paginato)
const { items, page_info } = await client.listAssets({ page: 1, limit: 20 });

// singolo
const asset = await client.getAsset("call_001");

// elimina
await client.deleteAsset("call_001");
```

```ts
type Asset = {
  asset_id: string;
  asset_class?: string;
  media_type: "audio" | "video";
  source_type: "url" | "s3" | "file";
  types: Array<"audio" | "video" | "transcript">;
  status: "pending" | "indexing" | "ready" | "failed";
  metadata?: Record<string, string | number | boolean>;
  duration_ms?: number;
  created_at: string;
  updated_at: string;
};
```

## Errori

```ts
import { SfVoiceMediaError } from "@sf-voice/media";

try {
  await client.search({ query: "...", asset_class: "customer_acme" });
} catch (err) {
  if (err instanceof SfVoiceMediaError) {
    console.error(err.code);    // codice leggibile dalla macchina
    console.error(err.status);  // stato HTTP
    console.error(err.message); // leggibile dall'utente
  }
}
```

| Classe | Quando |
| --- | --- |
| `SfVoiceMediaError` | Risposta API non-2xx. |
| `SfVoiceMediaRequestTimeoutError` | Una singola richiesta ha superato `timeoutMs`. |
| `SfVoiceMediaPollTimeoutError` | `pollTask` è scaduto prima dello stato terminale. Porta `taskId`. |
