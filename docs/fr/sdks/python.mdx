`sf-voice-media` — basé sur `httpx`. Clients synchrone et asynchrone avec des interfaces identiques.

Utilisez le client synchrone pour les scripts et les workers. Utilisez le client asynchrone dans FastAPI, Starlette ou d'autres services async.

## Installation

```sh
pip install sf-voice-media==0.1.1
```

Python 3.9+. Dépendance : `httpx >= 0.27`.

## Client synchrone

```python
from sf_voice import SfVoiceMedia

client = SfVoiceMedia(
    api_key="your_api_key",
    base_url="https://api.sf-voice.com",
)

# à utiliser comme gestionnaire de contexte pour fermer automatiquement le pool de connexions
with SfVoiceMedia(api_key="...") as client:
    ...
```

## Client asynchrone

```python
from sf_voice import AsyncSfVoiceMedia
import asyncio

async def main():
    async with AsyncSfVoiceMedia(api_key="...") as client:
        ...

asyncio.run(main())
```

Le client asynchrone a les mêmes noms de méthodes. Chaque méthode est une coroutine.

## Ingestion

```python
# depuis une URL
resp = client.ingest(
    source="url",
    asset_id="call_001",
    asset_class="customer_acme",
    url="https://storage.example.com/calls/001.mp3",
    media_type="audio",
    types=["audio", "transcript"],
    metadata={"caller": "+14155550199"},
)
print(resp.task_id)

# depuis un fichier
with open("call.mp3", "rb") as f:
    resp = client.ingest(
        source="file",
        asset_id="call_001",
        asset_class="customer_acme",
        file=f.read(),
        filename="call.mp3",
        content_type="audio/mpeg",
        media_type="audio",
        types=["audio", "transcript"],
    )
```

## Attendre l'indexation

```python
task = client.poll_task(
    resp.task_id,
    interval_s=2.0,
    timeout_s=120.0,
)

if task.status == "failed":
    raise RuntimeError(task.error or "indexing failed")
```

`poll_task` lève `SfVoiceMediaPollTimeoutError` si la tâche ne se termine pas dans `timeout_s`.

## Recherche

```python
result = client.search(
    query="customer asks about pricing",
    asset_class="customer_acme",
    types=["transcript"],
    threshold=0.7,
    limit=10,
)

for r in result.results:
    print(r.asset_id, r.start_ms, r.end_ms, r.score)
```

Recherche par assets spécifiques :

```python
result = client.search(
    query="cancellation request",
    asset_ids=["call_001", "call_002"],
)
```

## Assets

```python
# liste
resp = client.list_assets(page=1, limit=20)
for asset in resp.items:
    print(asset.asset_id, asset.status)

# unitaire
asset = client.get_asset("call_001")

# supprimer
client.delete_asset("call_001")
```

## Erreurs

```python
from sf_voice import SfVoiceMedia, SfVoiceMediaError, SfVoiceMediaPollTimeoutError

try:
    task = client.poll_task(task_id, timeout_s=60.0)
except SfVoiceMediaPollTimeoutError as e:
    print(f"timed out waiting for {e.task_id}")
except SfVoiceMediaError as e:
    print(e.code, e.status, e.message)
```

| Exception | Quand |
| --- | --- |
| `SfVoiceMediaError` | Réponse API non-2xx. Porte `code`, `message`, `status`. |
| `SfVoiceMediaRequestTimeoutError` | La requête HTTP a expiré. Porte `timeout_ms`. |
| `SfVoiceMediaPollTimeoutError` | `poll_task` a expiré. Porte `task_id`, `timeout_ms`. |
