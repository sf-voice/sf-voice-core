`@sf-voice/media` — nécessite un runtime disposant de `fetch`, `FormData` et `Blob`.

Version actuelle : `0.1.1`. Épinglez-la tant que l'API évolue.

## Installation

<CodeGroup>
```sh pnpm
pnpm add @sf-voice/media@0.1.1
```

```sh npm
npm install @sf-voice/media@0.1.1
```

```sh bun
bun add @sf-voice/media@0.1.1
```
</CodeGroup>

## Créer un client

```ts
import { SfVoiceMedia } from "@sf-voice/media";

const client = new SfVoiceMedia({
  baseUrl: "https://api.sf-voice.com",
  apiKey: process.env.SF_VOICE_API_KEY!,
});
```

| Option | Type | Défaut | |
| --- | --- | --- | --- |
| `baseUrl` | `string` | — | Obligatoire. |
| `apiKey` | `string` | — | Obligatoire. Envoyé en tant que `X-API-Key`. |
| `timeoutMs` | `number` | `30_000` | Délai d'abandon par requête. |

## Ingestion

`ingest(request)` envoie un enregistrement et retourne immédiatement un `task_id`.

### Depuis une URL

```ts
const { task_id } = await client.ingest({
  source: "url",
  asset_id: "call_001",
  asset_class: "customer_acme",
  url: "https://storage.example.com/calls/001.mp3",
  media_type: "audio",
  types: ["audio", "transcript"],
  metadata: { caller: "+14155550199" },
});
```

### Depuis S3

```ts
const { task_id } = await client.ingest({
  source: "s3",
  asset_id: "call_001",
  asset_class: "customer_acme",
  s3_key: "calls/customer_acme/001.mp3",
  media_type: "audio",
  types: ["audio", "transcript"],
});
```

### Depuis un fichier

```ts
const buffer = await file.arrayBuffer();

const { task_id } = await client.ingest({
  source: "file",
  asset_id: "call_001",
  asset_class: "customer_acme",
  file: buffer,
  filename: "001.mp3",
  content_type: "audio/mpeg",
  media_type: "audio",
  types: ["audio", "transcript"],
});
```

`file` accepte `Blob`, `ArrayBuffer` ou `Uint8Array`.

### Champs communs

| Champ | Obligatoire | |
| --- | --- | --- |
| `asset_id` | Oui | Votre identifiant. Renvoyé dans les résultats de recherche. |
| `asset_class` | Non | Groupe logique. Limite la recherche pour que les résultats ne fuient pas au-delà des frontières. |
| `media_type` | Non | `"audio"` ou `"video"`. Détecté automatiquement s'il est omis. |
| `types` | Non | Surfaces à indexer : `"audio"`, `"video"`, `"transcript"`. Toutes par défaut. |
| `metadata` | Non | Paires plates string/number/boolean pour votre usage. |

### Réponse

```ts
type IngestResponse = {
  asset_id: string;
  task_id: string;
  status: "pending";
};
```

## Tâches

### Interroger jusqu'à ce que la tâche soit prête

```ts
const task = await client.pollTask(task_id, {
  intervalMs: 2_000,   // fréquence des vérifications (par défaut 1_500)
  timeoutMs: 120_000,  // abandon après ce délai (par défaut 120_000)
});

if (task.status === "failed") {
  throw new Error(task.error ?? "indexing failed");
}
```

`pollTask` lève `SfVoiceMediaPollTimeoutError` si la tâche n'atteint pas `ready` ou `failed` dans le délai imparti.

### Vérifier une seule fois

```ts
const task = await client.getTask(task_id);
// task.status: "pending" | "indexing" | "ready" | "failed"
```

### Forme de la tâche

```ts
type Task = {
  task_id: string;
  asset_id: string;
  asset_class?: string;
  types: Array<"audio" | "video" | "transcript">;
  status: "pending" | "indexing" | "ready" | "failed";
  error?: string;
  created_at: string;
  completed_at?: string;
};
```

## Recherche

`search(request)` renvoie des correspondances horodatées dans les enregistrements indexés.

### Par asset class

```ts
const { results } = await client.search({
  query: "customer asks about cancelling their subscription",
  asset_class: "customer_acme",
  types: ["transcript"],
  threshold: 0.7,
  limit: 10,
});
```

### Par assets spécifiques

```ts
const { results } = await client.search({
  query: "pricing question",
  asset_ids: ["call_001", "call_002"],
});
```

### Rechercher dans tous les assets indexés

```ts
const { results } = await client.search({
  query: "AI interrupted the caller",
  scope: "all",
});
```

<Warning>
  `scope: "all"` parcourt l'ensemble de vos assets indexés. Utilisez `asset_class` ou `asset_ids` pour tout ce qui est destiné aux clients.
</Warning>

### Champs de recherche

| Champ | Obligatoire | |
| --- | --- | --- |
| `query` | Oui | Langage naturel. |
| `types` | Non | `"audio"`, `"video"`, `"transcript"`. Tous si omis. |
| `asset_class` | Non | Restreindre à un groupe logique. |
| `asset_ids` | Non | Restreindre à un ensemble d'assets spécifiques. |
| `scope` | Non | Définir à `"all"` pour rechercher dans tous les assets indexés. |
| `threshold` | Non | `0.0`–`1.0`. Par défaut `0.5`. Plus haut = moins de résultats, plus fiables. |
| `page` | Non | Numéro de page. |
| `limit` | Non | Résultats par page. Max 50. |

### Forme du résultat

```ts
type SearchResult = {
  asset_id: string;
  score: number;      // 0.0–1.0
  start_ms: number;   // décalage dans l'enregistrement
  end_ms: number;
  match_type: "audio" | "video" | "transcript";
  thumbnail_url?: string;
};
```

`start_ms` et `end_ms` sont des millisecondes dans l'enregistrement source. Utilisez-les pour positionner un lecteur sur le segment correspondant.

## Assets

```ts
// liste (paginée)
const { items, page_info } = await client.listAssets({ page: 1, limit: 20 });

// unitaire
const asset = await client.getAsset("call_001");

// supprimer
await client.deleteAsset("call_001");
```

```ts
type Asset = {
  asset_id: string;
  asset_class?: string;
  media_type: "audio" | "video";
  source_type: "url" | "s3" | "file";
  types: Array<"audio" | "video" | "transcript">;
  status: "pending" | "indexing" | "ready" | "failed";
  metadata?: Record<string, string | number | boolean>;
  duration_ms?: number;
  created_at: string;
  updated_at: string;
};
```

## Erreurs

```ts
import { SfVoiceMediaError } from "@sf-voice/media";

try {
  await client.search({ query: "...", asset_class: "customer_acme" });
} catch (err) {
  if (err instanceof SfVoiceMediaError) {
    console.error(err.code);    // code lisible par machine
    console.error(err.status);  // statut HTTP
    console.error(err.message); // lisible par humain
  }
}
```

| Classe | Quand |
| --- | --- |
| `SfVoiceMediaError` | Réponse API non-2xx. |
| `SfVoiceMediaRequestTimeoutError` | Une requête unique a dépassé `timeoutMs`. |
| `SfVoiceMediaPollTimeoutError` | `pollTask` a expiré avant un état terminal. Porte `taskId`. |
