`@sf-voice/media` — requiere un runtime con `fetch`, `FormData` y `Blob`.

Versión actual: `0.1.1`. Fíjela mientras la API esté en movimiento.

## Instalación

<CodeGroup>
```sh pnpm
pnpm add @sf-voice/media@0.1.1
```

```sh npm
npm install @sf-voice/media@0.1.1
```

```sh bun
bun add @sf-voice/media@0.1.1
```
</CodeGroup>

## Crear un cliente

```ts
import { SfVoiceMedia } from "@sf-voice/media";

const client = new SfVoiceMedia({
  baseUrl: "https://api.sf-voice.com",
  apiKey: process.env.SF_VOICE_API_KEY!,
});
```

| Opción | Tipo | Por defecto | |
| --- | --- | --- | --- |
| `baseUrl` | `string` | — | Requerido. |
| `apiKey` | `string` | — | Requerido. Se envía como `X-API-Key`. |
| `timeoutMs` | `number` | `30_000` | Tiempo de aborto por petición. |

## Ingesta

`ingest(request)` envía una grabación y devuelve un `task_id` de inmediato.

### Desde una URL

```ts
const { task_id } = await client.ingest({
  source: "url",
  asset_id: "call_001",
  asset_class: "customer_acme",
  url: "https://storage.example.com/calls/001.mp3",
  media_type: "audio",
  types: ["audio", "transcript"],
  metadata: { caller: "+14155550199" },
});
```

### Desde S3

```ts
const { task_id } = await client.ingest({
  source: "s3",
  asset_id: "call_001",
  asset_class: "customer_acme",
  s3_key: "calls/customer_acme/001.mp3",
  media_type: "audio",
  types: ["audio", "transcript"],
});
```

### Desde un archivo

```ts
const buffer = await file.arrayBuffer();

const { task_id } = await client.ingest({
  source: "file",
  asset_id: "call_001",
  asset_class: "customer_acme",
  file: buffer,
  filename: "001.mp3",
  content_type: "audio/mpeg",
  media_type: "audio",
  types: ["audio", "transcript"],
});
```

`file` acepta `Blob`, `ArrayBuffer` o `Uint8Array`.

### Campos comunes

| Campo | Requerido | |
| --- | --- | --- |
| `asset_id` | Sí | Su ID. Se devuelve en los resultados de búsqueda. |
| `asset_class` | No | Grupo lógico. Limita la búsqueda para que los resultados no se filtren entre fronteras. |
| `media_type` | No | `"audio"` o `"video"`. Se detecta automáticamente si se omite. |
| `types` | No | Superficies a indexar: `"audio"`, `"video"`, `"transcript"`. Por defecto todas. |
| `metadata` | No | Pares planos string/number/boolean para su uso. |

### Respuesta

```ts
type IngestResponse = {
  asset_id: string;
  task_id: string;
  status: "pending";
};
```

## Tareas

### Sondear hasta que esté lista

```ts
const task = await client.pollTask(task_id, {
  intervalMs: 2_000,   // cada cuánto comprobar (por defecto 1_500)
  timeoutMs: 120_000,  // rendirse después de este tiempo (por defecto 120_000)
});

if (task.status === "failed") {
  throw new Error(task.error ?? "indexing failed");
}
```

`pollTask` lanza `SfVoiceMediaPollTimeoutError` si la tarea no alcanza `ready` o `failed` dentro del timeout.

### Comprobar una sola vez

```ts
const task = await client.getTask(task_id);
// task.status: "pending" | "indexing" | "ready" | "failed"
```

### Forma de la tarea

```ts
type Task = {
  task_id: string;
  asset_id: string;
  asset_class?: string;
  types: Array<"audio" | "video" | "transcript">;
  status: "pending" | "indexing" | "ready" | "failed";
  error?: string;
  created_at: string;
  completed_at?: string;
};
```

## Búsqueda

`search(request)` devuelve coincidencias con marcas de tiempo dentro de las grabaciones indexadas.

### Por asset class

```ts
const { results } = await client.search({
  query: "customer asks about cancelling their subscription",
  asset_class: "customer_acme",
  types: ["transcript"],
  threshold: 0.7,
  limit: 10,
});
```

### Por assets específicos

```ts
const { results } = await client.search({
  query: "pricing question",
  asset_ids: ["call_001", "call_002"],
});
```

### Buscar en todos los assets indexados

```ts
const { results } = await client.search({
  query: "AI interrupted the caller",
  scope: "all",
});
```

<Warning>
  `scope: "all"` busca en todos sus assets indexados. Use `asset_class` o `asset_ids` para cualquier cosa orientada al cliente.
</Warning>

### Campos de búsqueda

| Campo | Requerido | |
| --- | --- | --- |
| `query` | Sí | Lenguaje natural. |
| `types` | No | `"audio"`, `"video"`, `"transcript"`. Si se omite, busca en todos. |
| `asset_class` | No | Restringir a un grupo lógico. |
| `asset_ids` | No | Restringir a un conjunto específico de assets. |
| `scope` | No | Establezca `"all"` para buscar en los assets indexados. |
| `threshold` | No | `0.0`–`1.0`. Por defecto `0.5`. Más alto = menos resultados, más fiables. |
| `page` | No | Número de página. |
| `limit` | No | Resultados por página. Máx. 50. |

### Forma del resultado

```ts
type SearchResult = {
  asset_id: string;
  score: number;      // 0.0–1.0
  start_ms: number;   // desplazamiento dentro de la grabación
  end_ms: number;
  match_type: "audio" | "video" | "transcript";
  thumbnail_url?: string;
};
```

`start_ms` y `end_ms` son milisegundos dentro de la grabación fuente. Úselos para colocar un reproductor en el segmento coincidente.

## Assets

```ts
// listar (paginado)
const { items, page_info } = await client.listAssets({ page: 1, limit: 20 });

// uno
const asset = await client.getAsset("call_001");

// borrar
await client.deleteAsset("call_001");
```

```ts
type Asset = {
  asset_id: string;
  asset_class?: string;
  media_type: "audio" | "video";
  source_type: "url" | "s3" | "file";
  types: Array<"audio" | "video" | "transcript">;
  status: "pending" | "indexing" | "ready" | "failed";
  metadata?: Record<string, string | number | boolean>;
  duration_ms?: number;
  created_at: string;
  updated_at: string;
};
```

## Errores

```ts
import { SfVoiceMediaError } from "@sf-voice/media";

try {
  await client.search({ query: "...", asset_class: "customer_acme" });
} catch (err) {
  if (err instanceof SfVoiceMediaError) {
    console.error(err.code);    // código legible por máquina
    console.error(err.status);  // estado HTTP
    console.error(err.message); // legible por humanos
  }
}
```

| Clase | Cuándo |
| --- | --- |
| `SfVoiceMediaError` | Respuesta de la API distinta de 2xx. |
| `SfVoiceMediaRequestTimeoutError` | Una sola petición superó `timeoutMs`. |
| `SfVoiceMediaPollTimeoutError` | `pollTask` superó el tiempo antes del estado terminal. Lleva `taskId`. |
