`@sf-voice/media` —— 需要具备 `fetch`、`FormData` 和 `Blob` 的运行时。

当前版本：`0.1.1`。在 API 变化期间请固定该版本。

## 安装

<CodeGroup>
```sh pnpm
pnpm add @sf-voice/media@0.1.1
```

```sh npm
npm install @sf-voice/media@0.1.1
```

```sh bun
bun add @sf-voice/media@0.1.1
```
</CodeGroup>

## 创建客户端

```ts
import { SfVoiceMedia } from "@sf-voice/media";

const client = new SfVoiceMedia({
  baseUrl: "https://api.sf-voice.com",
  apiKey: process.env.SF_VOICE_API_KEY!,
});
```

| 选项 | 类型 | 默认值 | |
| --- | --- | --- | --- |
| `baseUrl` | `string` | — | 必填。 |
| `apiKey` | `string` | — | 必填。作为 `X-API-Key` 发送。 |
| `timeoutMs` | `number` | `30_000` | 单次请求的中断超时。 |

## 摄入

`ingest(request)` 提交一条录音并立即返回 `task_id`。

### 从 URL

```ts
const { task_id } = await client.ingest({
  source: "url",
  asset_id: "call_001",
  asset_class: "customer_acme",
  url: "https://storage.example.com/calls/001.mp3",
  media_type: "audio",
  types: ["audio", "transcript"],
  metadata: { caller: "+14155550199" },
});
```

### 从 S3

```ts
const { task_id } = await client.ingest({
  source: "s3",
  asset_id: "call_001",
  asset_class: "customer_acme",
  s3_key: "calls/customer_acme/001.mp3",
  media_type: "audio",
  types: ["audio", "transcript"],
});
```

### 从文件

```ts
const buffer = await file.arrayBuffer();

const { task_id } = await client.ingest({
  source: "file",
  asset_id: "call_001",
  asset_class: "customer_acme",
  file: buffer,
  filename: "001.mp3",
  content_type: "audio/mpeg",
  media_type: "audio",
  types: ["audio", "transcript"],
});
```

`file` 接受 `Blob`、`ArrayBuffer` 或 `Uint8Array`。

### 通用字段

| 字段 | 必填 | |
| --- | --- | --- |
| `asset_id` | 是 | 你的 ID。会在搜索结果中返回。 |
| `asset_class` | 否 | 逻辑分组。限制搜索范围，使结果不会跨越边界泄漏。 |
| `media_type` | 否 | `"audio"` 或 `"video"`。省略时会自动检测。 |
| `types` | 否 | 要索引的表面：`"audio"`、`"video"`、`"transcript"`。默认全部。 |
| `metadata` | 否 | 供你自己使用的扁平 string/number/boolean 键值对。 |

### 响应

```ts
type IngestResponse = {
  asset_id: string;
  task_id: string;
  status: "pending";
};
```

## 任务

### 轮询直到完成

```ts
const task = await client.pollTask(task_id, {
  intervalMs: 2_000,   // 检查间隔（默认 1_500）
  timeoutMs: 120_000,  // 超过此时长后放弃（默认 120_000）
});

if (task.status === "failed") {
  throw new Error(task.error ?? "indexing failed");
}
```

如果任务未能在超时时间内到达 `ready` 或 `failed`，`pollTask` 会抛出 `SfVoiceMediaPollTimeoutError`。

### 检查一次

```ts
const task = await client.getTask(task_id);
// task.status: "pending" | "indexing" | "ready" | "failed"
```

### 任务结构

```ts
type Task = {
  task_id: string;
  asset_id: string;
  asset_class?: string;
  types: Array<"audio" | "video" | "transcript">;
  status: "pending" | "indexing" | "ready" | "failed";
  error?: string;
  created_at: string;
  completed_at?: string;
};
```

## 搜索

`search(request)` 在已索引的录音中返回带时间戳的匹配。

### 按 asset class 搜索

```ts
const { results } = await client.search({
  query: "customer asks about cancelling their subscription",
  asset_class: "customer_acme",
  types: ["transcript"],
  threshold: 0.7,
  limit: 10,
});
```

### 按指定资产搜索

```ts
const { results } = await client.search({
  query: "pricing question",
  asset_ids: ["call_001", "call_002"],
});
```

### 在所有已索引的资产中搜索

```ts
const { results } = await client.search({
  query: "AI interrupted the caller",
  scope: "all",
});
```

<Warning>
  `scope: "all"` 会在你所有已索引的资产中搜索。任何面向客户的场景请使用 `asset_class` 或 `asset_ids`。
</Warning>

### 搜索字段

| 字段 | 必填 | |
| --- | --- | --- |
| `query` | 是 | 自然语言。 |
| `types` | 否 | `"audio"`、`"video"`、`"transcript"`。省略时全部搜索。 |
| `asset_class` | 否 | 限定到一个逻辑分组。 |
| `asset_ids` | 否 | 限定到特定的一组资产。 |
| `scope` | 否 | 设置为 `"all"` 以在所有已索引资产中搜索。 |
| `threshold` | 否 | `0.0`–`1.0`。默认 `0.5`。值越高，结果越少但越可信。 |
| `page` | 否 | 页码。 |
| `limit` | 否 | 每页结果数。最大 50。 |

### 结果结构

```ts
type SearchResult = {
  asset_id: string;
  score: number;      // 0.0–1.0
  start_ms: number;   // 录音中的偏移量
  end_ms: number;
  match_type: "audio" | "video" | "transcript";
  thumbnail_url?: string;
};
```

`start_ms` 和 `end_ms` 是源录音中的毫秒数。用它们将播放器定位到匹配片段。

## 资产

```ts
// 列表（分页）
const { items, page_info } = await client.listAssets({ page: 1, limit: 20 });

// 单个
const asset = await client.getAsset("call_001");

await client.deleteAsset("call_001");
```

```ts
type Asset = {
  asset_id: string;
  asset_class?: string;
  media_type: "audio" | "video";
  source_type: "url" | "s3" | "file";
  types: Array<"audio" | "video" | "transcript">;
  status: "pending" | "indexing" | "ready" | "failed";
  metadata?: Record<string, string | number | boolean>;
  duration_ms?: number;
  created_at: string;
  updated_at: string;
};
```

## 错误

```ts
import { SfVoiceMediaError } from "@sf-voice/media";

try {
  await client.search({ query: "...", asset_class: "customer_acme" });
} catch (err) {
  if (err instanceof SfVoiceMediaError) {
    console.error(err.code);    // 机器可读的代码
    console.error(err.status);  // HTTP 状态
    console.error(err.message); // 面向用户的可读信息
  }
}
```

| 类 | 何时 |
| --- | --- |
| `SfVoiceMediaError` | API 返回非 2xx 响应。 |
| `SfVoiceMediaRequestTimeoutError` | 单次请求超过了 `timeoutMs`。 |
| `SfVoiceMediaPollTimeoutError` | `pollTask` 在到达终态前超时。携带 `taskId`。 |
